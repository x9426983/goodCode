开始选择数据结构
↓
需要存储什么类型的数据？
↓
├── 需要顺序访问吗？
│   ↓
│   ├── 是 → 需要随机访问吗？
│   │   ↓
│   │   ├── 是 → 数组/ArrayList
│   │   ↓
│   │   └── 否 → 需要频繁插入删除吗？
│   │       ↓
│   │       ├── 是 → LinkedList
│   │       ↓
│   │       └── 否 → 根据LIFO/FIFO选择栈/队列
│   │
│   └── 否 → 需要快速查找吗？
│       ↓
│       ├── 是 → 需要键值对吗？
│       │   ↓
│       │   ├── 是 → 需要有序吗？
│       │   │   ↓
│       │   │   ├── 是 → TreeMap
│       │   │   ↓
│       │   │   └── 否 → HashMap
│       │   ↓
│       │   └── 否 → 需要唯一性吗？
│       │       ↓
│       │       ├── 是 → 需要有序吗？
│       │       │   ↓
│       │       │   ├── 是 → TreeSet
│       │       │   ↓
│       │       │   └── 否 → HashSet
│       │       ↓
│       │       └── 否 → 考虑其他需求
│       ↓
│       └── 否 → 有层次关系吗？
│           ↓
│           ├── 是 → 需要优先级吗？
│           │   ↓
│           │   ├── 是 → 堆/PriorityQueue
│           │   ↓
│           │   └── 否 → 二叉树/多叉树
│           ↓
│           └── 否 → 有多对多关系吗？
│               ↓
│               ├── 是 → 图结构
│               ↓
│               └── 否 → 重新分析需求
