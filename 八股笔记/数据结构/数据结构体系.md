数据结构体系
├── 线性结构 (一对一)
│   ├── 顺序存储
│   │   ├── 数组 (Array)
│   │   ├── 动态数组 (ArrayList/Vector)
│   │   └── 栈 (Stack - 数组实现)
│   │
│   └── 链式存储  
│       ├── 单向链表 (Singly Linked List)
│       ├── 双向链表 (Doubly Linked List)
│       ├── 循环链表 (Circular Linked List)
│       ├── 栈 (Stack - 链表实现)
│       └── 队列 (Queue - 链表实现)
│
├── 树形结构 (一对多)
│   ├── 二叉树 (Binary Tree)
│   │   ├── 满二叉树 (Full Binary Tree)
│   │   ├── 完全二叉树 (Complete Binary Tree)
│   │   ├── 二叉搜索树 (BST - Binary Search Tree)
│   │   │   ├── 平衡二叉搜索树
│   │   │   │   ├── AVL树
│   │   │   │   └── 红黑树 (Red-Black Tree)
│   │   │   └── 非平衡BST
│   │   │
│   │   └── 堆 (Heap)
│   │       ├── 最小堆 (Min Heap)
│   │       └── 最大堆 (Max Heap)
│   │
│   ├── 多路搜索树
│   │   ├── B树 (B-Tree)
│   │   ├── B+树 (B+ Tree)
│   │   └── B*树
│   │
│   └── 特殊树
│       ├── Trie树 (字典树/前缀树)
│       ├── 哈夫曼树 (Huffman Tree)
│       └── 线段树 (Segment Tree)
│
├── 图形结构 (多对多)
│   ├── 按方向
│   │   ├── 有向图 (Directed Graph/Digraph)
│   │   └── 无向图 (Undirected Graph)
│   │
│   ├── 按权值
│   │   ├── 无权图
│   │   └── 带权图 (Weighted Graph)
│   │
│   └── 特殊图
│       ├── 树 (Tree - 无环连通图)
│       ├── 森林 (Forest - 多棵树)
│       └── 有向无环图 (DAG)
│
└── 集合结构 (无关系)
├── 集合 (Set)
│   ├── 无序集合 (基于哈希表)
│   │   ├── 哈希集合 (HashSet)
│   │   └── 链式哈希集合 (LinkedHashSet)
│   │
│   └── 有序集合 (基于搜索树)
│       └── 树集合 (TreeSet)
│
└── 映射 (Map)
├── 无序映射 (基于哈希表)
│   ├── 哈希映射 (HashMap)
│   └── 链式哈希映射 (LinkedHashMap)
│
└── 有序映射 (基于搜索树)
└── 树映射 (TreeMap)
