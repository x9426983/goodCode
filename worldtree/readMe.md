### 名词说明
场景 ：代表业务某个具体场景或一类场景，比如闪购频道页商家列表。一般作为接口调用的参数和调用方交互。
图 ：场景具体的编排逻辑。每个场景可以有多个不同名称的图，根据实验走不同图的逻辑。
图版本 ：同一个名称的图的每次修改提交会生成一个新的版本。对于同一张图，业务服务一台机器上只会使用一个版本。通过版本来控制图策略修改的下发和回滚。
算子 ：内聚的业务逻辑以同一接口实现，是图逻辑编排的基本单元。一个算子可以在一张图中多次使用，也可以在不同图中使用。
